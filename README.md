# Common device configuration for Xiaomi Taro family

## Currently supported SoCs

- SM8450 (taro)
- SM8475 (cape)
- SM7475 (ukee)

## Currently supported devices:

- Xiaomi 12 (cupid)
- Xiaomi 12S Pro (unicorn)
- Redmi K50 Ultra / Xiaomi 12T Pro (diting)
- POCO F5 / Redmi Note 12 Turbo (marble)
